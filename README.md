# AssaultCube-ESP-Python-Cheat

This project is an ESP (Extra Sensory Perception) cheat for the game AssaultCube. 

It provides an overlay that draws over entity players.

## Features

- Player ESP
- Health Bars
- Team Indicators
- Line of Sight
- FPS Counter

## Requirements

- Python 3.9+
- Required Libraries:
  - PyQt5
  - pymem
  - ctypes
 
``pip install PyQt5 pymem``
